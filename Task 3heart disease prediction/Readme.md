Task 3: Heart Disease Prediction

Author:
Muhammad Ahmad

Objective:
The objective is to build a machine learning model to predict whether a person is at risk of heart disease.

Dataset:
- Heart Disease Dataset (UCI/Kaggle)
- Features include:
  - Age
  - Sex
  - Blood Pressure
  - Cholesterol
  - etc.

Steps Performed:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training using Logistic Regression / Decision Tree

Model Used:
- Logistic Regression (or Decision Tree)

Evaluation Metrics:
- Accuracy
- Confusion Matrix
- ROC Curve

Results:
- Model achieved good accuracy
- Confusion matrix showed correct classification of most cases

Key Insights:
- Age and cholesterol are important factors
- Some features strongly influence prediction

Conclusion:
The model successfully predicts heart disease risk with reasonable accuracy.
