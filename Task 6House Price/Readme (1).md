Task 6: House Price Prediction

Author:
Muhammad Ahmad

Objective:
To predict house prices using features such as area, bedrooms, and condition using regression techniques.

Dataset:
- House Price Dataset (Kaggle)
- Features include:
  - Bedrooms
  - Living area (living_in_m2)
  - Grade
  - Condition
  - Basement
  - etc.

Steps Performed:
- Loaded dataset using pandas
- Data preprocessing and cleaning
- Feature selection
- Split data into training and testing sets

Model Used:
- Linear Regression

Evaluation Metrics:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

Results:
- Model predicts house prices with reasonable accuracy
- Predicted values closely follow actual values

Key Insights:
- Living area has strong impact on price
- Condition and grade also influence pricing
- More features can improve model performance

Conclusion:
The regression model successfully predicts house prices based on given features.
